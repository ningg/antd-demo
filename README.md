## 启动方法

本地开发联调,几个基本步骤:

````
# 在当前目录下, 安装 node_modules
npm install

# Note: 必要的时候, 先删除当前目录的 node_modules, 重新下载
rm -rf node_modules

npm install

# 本地编译
npm run build

# 以开发联调模式启动
npm run dev
````
